# motion-detection-OpenCV
This project detects real-time motion using a webcam and highlights movement with bounding boxes.

## Technologies Used
- Python
- OpenCV
- Imutils

## How to Run
1. Install dependencies:
   pip install opencv-python imutils
2. Run the program:
   python motion_detection.py

## Features
- Real-time camera motion detection
- Green bounding box for movement
- Status display (Normal / Movement Detected)
